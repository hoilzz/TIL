# 1. DP

## fibonacci

```
int fib(int n) {
  if (n==1 | n==2) {
    return 1;
  }
  return fib(n-2) + fib(n-1);
}
```

![img]('./img/2-1-1.png')

많은 계산이 중복됨.

1. top-down
중간 계산 결과를 caching 하여 중복 계산 피함.

2. bottom-up
f[1], f[2], ...f[n] 까지 순서대로 계산하여 중복 피함.
- 예컨대 f[8]을 계산하기 위해 f[6]+f[7] 임
- f[6]을 계산하기 위해 f[5]+f[4], ...
- 기본적인 값부터 계산하여 원하는 계산까지 도달
