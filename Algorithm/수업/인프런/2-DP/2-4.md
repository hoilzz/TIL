# matrix-chain Multiplication

행렬의 곱셈.

## pxq 행렬 A와 qxr 행렬 B 곱하기.

```C
void product(int A[][], int B[][], int C[][]) {
  for(int i = 0; i<p; i++) {
    for(int j = 0; j<r; j++) {
      C[i][j] = 0;
      for(int k=0; k<q; k++) {
        C[i][j] += A[i][k]*B[k][j];
      }
    }
  }
}
```

곱셈연산의 횟수 = pqr
