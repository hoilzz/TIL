# percentage

많은 CSS properties는 `%` 값을 가진다. (**가끔 parent object 기준으로 사이즈를 정의 하기 위해**). Many length properties 퍼센트를 사용한다.
- **`width`**
- **`margin`**
- **`padding`**
- **`font-size`** (where the size of the text is directly related to the size of its parent)
