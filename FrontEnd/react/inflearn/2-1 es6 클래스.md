# 자바스크립트의 클래스란
mdn에서 es6 에서 본다
- 생성자 메서드 있다
- 자바스크립트 클래스 안에서는 메서드만 만들 수 있다
- 변수 사용시 이런식으로 이니셜라이징
- 클래스 사용시 클래스를 꼭 선언해야함
- 클래스 선언전에 클래스 사용하면 reference error 발생
- 클래스 선언의 다양한 방법
- static 메서드도 생성 가능
- static은 객체로 만들기 전에도 사용가능
- extend를 통한 상속도 가능
- super 키워드를 통해 par2ent 클래스 호출 가능
